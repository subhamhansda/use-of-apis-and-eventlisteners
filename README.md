# use-of-apis-and-eventlisteners
This project consists of 4 mini-projects.
**The first project is to list all the colleges present in a country once the user types the country name and presses the submit button.
**In second one user will able to see one random dog image after clicking on the get image button.
**In third a random colour will be generated and fill the box ,with its colour code printed above.
**In fourth a random cat fact will generate once the user presses the get facts button.
